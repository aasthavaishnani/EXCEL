# Sales Performance Analytics Dashboard 📊

## 📝 Project Overview
This project is a comprehensive Data Analysis and Financial Forecasting tool developed using **Microsoft Excel 2019**. It transforms raw transactional data into actionable business insights through advanced statistical modeling, interactive visualizations, and automated reporting techniques.

The primary goal of this project was to analyze customer behavior, regional performance, and the financial impact of sales strategies using a data-driven approach.

---

## 🛠️ Key Technical Features

### 1. Advanced Statistical Analysis
- **Descriptive Statistics:** Generated a complete statistical summary (Mean, Median, Standard Deviation) to understand data distribution across transactions.
- **Data Integrity:** Performed extensive data cleaning, handling missing values and ensuring consistent formatting for over 200 records.

### 2. Financial Modeling & Forecasting
- **What-If Analysis:** Utilized Goal Seek and Data Tables to simulate various business scenarios and their impact on net revenue.
- **Trend Analysis:** Implemented manual calculations for percentage changes and shifts to identify growth patterns without relying on restricted automated functions.

### 3. Dynamic Business Intelligence Dashboard
- **Interactive Slicers & Timelines:** Connected multiple Pivot Tables to a single Slicer interface for real-time filtering by Region, Category, and Date.
- **KPI Scorecards:** Created high-level snapshots of Total Revenue, Total Orders, and Average Order Value using linked shapes for a professional UI.
- **Automated Timestamping:** Integrated a live NOW() function and custom VBA-based logic to track data updates and ensure report accuracy.

### 4. Advanced Lookup & Logic
- **Advanced Formulas:** Utilized complex INDEX(), MATCH(), and XLOOKUP() functions to retrieve specific customer and product data dynamically.
- **Conditional Formatting:** Applied rule-based formatting to highlight top-performing products and critical sales thresholds.

---

## 📂 Sheet Structure
- **Raw Data:** Contains cleaned transactional records with automated timestamps.
- **Analysis:** Dedicated space for Pivot Tables, ranking systems, and target achievement models.
- **Visualization:** Backend chart preparation and statistical summaries.
- **Dashboard:** The final interactive user interface featuring dynamic charts and control panels.

---

## 📈 Executive Summary of Insights
- **Top Performers:** Identified high-value customers and high-volume products within the dataset.
- **Regional Strength:** Analyzed sales distribution focused on the Gujarat region and other key territories.
- **Operational Efficiency:** Streamlined the reporting process from raw data entry to visual executive summaries.

## 🚀 How to Use
1. Open the **PR. Final Project.xlsx** file in Microsoft Excel 2019 or later.
2. Navigate to the **Dashboard** sheet to view the interactive interface.
3. Use the **Slicers and Timeline** on the left panel to filter data dynamically.
4. Review the **Analysis** sheet for detailed calculations and ranking metrics.
